# 图（Graph）

图的组成
- 顶点
- 边
- 边的权重
- 边的方向

树和链表也是一种图结构，树是一种没有回路的图
另外一种常见的米有回路的图结构是有向无环图，与树不同的是，有向无环图没有层级

图能干什么？
- Core Data 中的 data model
- 状态机

图的表示
- 邻接矩阵
- 邻接链表
